# House Price Prediction using Machine Learning

## Objective
To predict house prices based on area using Linear Regression.

## Dataset
The dataset contains house area and corresponding prices.

## Tools Used
- Python
- Pandas
- scikit-learn

## Steps Performed
- Loaded and explored the dataset
- Prepared input and output variables
- Trained a Linear Regression model
- Predicted house prices for new areas

## Result
The model successfully predicts house prices based on area.
